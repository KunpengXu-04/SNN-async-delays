# Presentation workspace

- `current/`: manually maintained current deck.  Do not overwrite it from a
  generation script.
- `generated/`: reproducible draft decks; review before merging.
- `archive/`: historical decks and backups, not evidence sources.
- `scripts/`: deck generation and explicit merge utilities.
- `assets/`: source images used by presentation scripts.
- `notes/`: presentation-specific notes, which may contain superseded claims.

The current deck is an editable communication artifact, not the scientific
source of truth.  Numerical claims must be checked against
`snn_async_delays/docs/CLAIMS_LEDGER.md` before presentation.
